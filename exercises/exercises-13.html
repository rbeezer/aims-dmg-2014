Exercise 1 This exercise asks you to verify a theorem. It will require you to become more adept manipulating the varieties and blocks of a design with Python and Sage.

  1. Construct the Witt design on \(23\) varieties using designs.WittDesign() in Sage. This is a \(4\)-design, though in this problem we shall consider it as a \(3\)-design, since a theorem tells us this happens.
  2. Compute the parameters of the design as a \(3\)-design.
  3. Compute every \(3\)-set and determine how many blocks each is contained in. If this is a design, the number should be the same for each \(3\)-set. Verify and compute this common number, which we usually refer to as \(\lambda\). Do not assume the above computation of the parameters is correct and merely check that value.

Full marks for a successful verification of the defining property of a design in the case of this example and a computation of the value of \(\lambda\).

Use Python and Sage to create the blocks of these two designs based on the Hadamard matrix of size 12, and check your work by providing your blocks as input to the BlockDesign() constructor.
